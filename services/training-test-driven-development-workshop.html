About the Workshop Training

This hands on workshop teaches Test Driven Development through practical exercises with C# and JavaScript. Learn the red green refactor cycle, testing pyramid concepts, and how TDD shapes code design and development practices. Developers gain skills to write meaningful tests, choose appropriate test levels, and maintain confidence as codebases grow, especially critical when using AI assisted development tools.

The workshop follows a learn demo practice cycle. Participants learn TDD principles, watch live demonstrations, then practice through guided exercises. Working in pairs or small groups encourages knowledge sharing and collaborative problem solving.

Tools like Visual Studio Code and testing frameworks illustrate techniques that adapt to any development environment. By the end, participants confidently apply TDD practices to real projects, preventing vibe coding from destroying project quality while leveraging AI for faster development.

Who is this course ideal for?

This workshop is designed for software developers, engineers, and tech leads who want to improve code quality, reduce bugs, and adopt Test Driven Development practices.

What are the technical prerequisites?

Participants should have VS Code installed and basic programming experience in C# or JavaScript. Familiarity with basic testing concepts is helpful but not required.

Do I need prior testing experience?

No prior testing experience is required. The workshop starts with fundamentals and progressively builds to advanced TDD techniques with plenty of practical exercises.

What is the workshop structure?

The full day workshop follows three phases: theoretical TDD foundations, live instructor demonstrations with xUnit/MochaJS/Puppeteer, and hands on participant practice with guided coding tasks in VS Code.
